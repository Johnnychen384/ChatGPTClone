# ChatGPT Clone

Welcome to the ChatGPT Clone project! This project aims to replicate the functionality of OpenAI's ChatGPT while using modern technologies such as Next.js, TypeScript, Firebase, and Tailwind CSS. Below, you'll find an overview of the project, its features, and how to access the deployed version.

## Features

- **Next.js 13**: Developed using Next.js 13 to leverage its advanced features for hybrid static and server rendering, enhancing SEO and performance.
- **TypeScript**: Strongly typed codebase to ensure better developer experience and maintainability.
- **Firebase Backend**: Utilizes Firebase for backend functionalities, including real-time database and authentication system for managing user sessions and securely storing chat histories.
- **Tailwind CSS**: Crafted with Tailwind CSS for a modern and responsive user interface adhering to mobile-first design principles.
- **CRUD Functionality**: Complete CRUD (Create, Read, Update, Delete) functionality allows users to create and save chat instances, retrieve chat logs, edit messages, and delete chat instances.
- **Deployment**: Application architecture is designed for deployment on Vercel, ensuring high availability and performance.

## Accessing the Deployed Version

The ChatGPT Clone project has been deployed to Vercel. You can access the deployed version by following this link: [ChatGPT Clone on Vercel](https://chat-gpt-clone-alpha-orcin.vercel.app/)
